    Aide & Informations

    Objectif : charger des données depuis Excel, puis analyser des tendances via des graphiques (ECharts) et des tableaux interactifs.

    Vue d’ensemble de l’interface
    • Header : actions globales (Ajouter section, Export sauvegarde, Import sauvegarde JSON, Reset total).
    • Sections : chaque section possède son graphique + ses filtres + ses données brutes (tableau).
    • Modales : saisie/édition des lignes, création/édition des sections, et aide.

    1) Importer les données (Excel .xlsx)
    • Par section : utilisez « 📁 Importer Excel » dans la section concernée.
    • Intérêt : vous pouvez importer plusieurs fichiers successivement, et le dashboard s’actualise.

    Format attendu (colonnes)
    • Colonnes obligatoires :
    - Type : doit correspondre à une section existante (ex: “Dispersion 14h”, ou nom d’une section personnalisée).
    - Date : Excel doit fournir une date interprétable (le script gère les dates Excel converties).
    • Sections standard : le script cherche aussi des colonnes qui contiennent “annonce” et “flash”.
    • Sections personnalisées : les colonnes attendues sont basées sur les champs configurés (noms exactement comme définis dans la section).

    Anti-doublons (important)
    • Si vous importez plusieurs fois le même couple (Type + Date), la ligne est dédupliquée.

    2) Graphiques (ECharts) — comment les lire
    • Le graphique utilise un axe X construit à partir du regroupement (Jour/Semaine/Mois/Année).
    • Le mode normal affiche des séries pour les champs de la section.
    • Le mode comparaison s’active dès que vous cochez plus d’une année :
    - une série par année et par champ
    - couleurs distinctes pour différencier rapidement les tendances.

    3) Filtres — ce qui change quand vous cliquez
    • Jour : affiche uniquement les lignes correspondant au jour sélectionné.
    • Mois : filtre sur le mois sélectionné.
    • Années :
    - cochez 1 année → mode normal
    - cochez plusieurs années → mode comparaison
    • Regroupement : change la “granularité” de l’axe X (et donc la forme de la tendance).
    • Tous les filtres s’appliquent à la fois au graphique et au tableau.

    4) Données brutes (tableau) — édition & suppression
    • Ouvrez « Données brutes — [nom de section] » pour voir les lignes.
    • Tri : l’en-tête Date permet de basculer asc / desc.
    • ✏️ Modifier : ouvre la modale de saisie avec les valeurs actuelles.
    - Pour une section standard : colonnes annoncées/flashées selon visibilité.
    - Pour une section personnalisée : champs dynamiques générés à partir de la configuration.
    • 🗑️ Supprimer : supprime la ligne puis force la mise à jour du graphique et du tableau.

    5) Export — conserver ou partager vos analyses
    • 📷 Exporter PNG : export direct du graphique en image (utile pour report / mail).
    • 📊 Exporter Excel : export des données de la section.

    6) Réinitialisation (Reset total)
    • « Réinitialiser toutes les données » : efface les données stockées dans le navigateur via localStorage.
    • Résultat : graphiques vidés + filtres recalibrés + sections personnalisées supprimées.

    7) Subtilités & comportements “cachés”
    • Persistant : les données importées restent après rechargement de page (localStorage).
    • Déduplication : clé métier = (Type + Date).
    • Sections personnalisées : champs dynamiques → import et édition utilisent exactement ces champs.
    • Clôture de la modale d’aide :
    - cliquez sur « × » ou « Fermer »
    - ou cliquez sur le fond (overlay) pour la fermer.
