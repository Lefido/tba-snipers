Aide et Informations

🚀 Bienvenue sur le Tableau de Bord

Cet outil permet d'analyser vos flux de colis via des graphiques dynamiques et des tableaux de données persistants (LocalStorage).

🛠️ Barre d'Actions Principale

  • Ajouter une section : Créez un nouveau graphique. Vous pouvez définir le type (Lignes/Barres) et créer des champs de données personnalisés avec leurs propres couleurs.
  • Exporter Sauvegarde : Télécharge un fichier .json contenant TOUTES vos sections et vos données. Idéal pour le partage ou le backup.
  • Importer Sauvegarde : Restaure votre dashboard à partir d'un fichier JSON précédemment exporté.
  • Réinitialiser : Efface l'intégralité des données stockées dans votre navigateur.

📊 Contrôles des Sections

  • Ajouter : Ouvre un formulaire pour saisir manuellement une nouvelle date et ses valeurs.
  • Importer Excel : Charge des données en masse. Le fichier doit contenir une colonne "Type" (nom de la section) et "Date".
  • Exporter Excel : Génère un tableur propre pour la section actuelle.
  • Exporter PNG : Enregistre le graphique affiché sous forme d'image.
  • Modifier : Permet de renommer la section ou de changer la configuration des champs.
  • / : Réorganise l'ordre d'affichage des sections sur votre écran.

💡 Astuces

Sous chaque graphique, le volet "Données brutes" vous permet de consulter l'historique complet. Utilisez les icônes de modification et de suppression pour corriger des erreurs de saisie.

Le menu Navigation dans le header vous permet de vous déplacer rapidement vers une section spécifique.
